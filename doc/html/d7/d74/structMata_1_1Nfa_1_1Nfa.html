Key value store for additional attributes for the NFA. Keys are attribute names as strings and the value types are up to the user. For example, we can set up attributes such as "state_dict" for state dictionary attribute mapping states to their respective names, or "transition_dict" for transition dictionary adding a human-readable meaning to each transition.
