Detailed Description

The following namespace contains methods which would be otherwise in anonymous namespace to make them accessible to users of library. Typically, that are different algorithms for operations such as complement, inclusion, or universality checking. In Nfa interface, there are dispatch functions calling these function according to parameters provided by a user.
